This folder contains the IBP plus LFM model code.
ibpmultigpCreate        - Creates the IBPLFM model.
ibpmultigpExpandParam   - Function to set params values.
ibpmultigpExtractParam  - Function to extract params values.
ibpmultigpComputeKernels- Compute covariance matrices.
ibpmultigpGradient      - Compute gradient for the model.
ibpmultigpLovalCovGradi - Compute gradient for kernel hyperparameters.
ibpmultigpLowerBound    - Computes the lower bound.
